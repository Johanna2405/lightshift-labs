# lightshift labs – Interactive Timeline Microsite

Dieses Projekt ist im Rahmen des Studiengangs Medieninformatik als Uni-Projekt entstanden. Ziel ist die Umsetzung einer interaktiven Firmenhistorie des fiktiven Indie-Tech-/Gaming-Studios **lightshift labs** als Microsite mit einer zentralen Timeline (2016–2025).

## Tech-Stack

- React (Vite)
- JavaScript
- Tailwind CSS
- Framer Motion
- `<model-viewer>` (Webkomponente für 3D-Modelle)
